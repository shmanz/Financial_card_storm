## 📊 카드 정보 이해하기

### 카드 예시: "지름신 소환"

```
┌─────────────────┐
│   💳 3          │ ← 코스트 (에너지)
│                 │
│  지름신 소환     │
│                 │
│  🛍️ SHOPPING   │
│                 │
│  [큰 이모지]    │
│                 │
│  설명:          │
│  5 피해를 줍니다│ ← 실제 피해량
│                 │
│  ⚔️ 5  🛡️ 0   │ ← 공격력/방어력
└─────────────────┘
```

**의미**:
- 코스트 3 소모
- 상대에게 **정확히 5 피해**
- 방어력 0 (공격 카드)

### 카드 종류

#### 공격 카드

```
⚔️ > 0, 🛡️ = 0
→ 상대 HP 감소
```

#### 방어 카드

```
⚔️ = 0, 🛡️ > 0
→ 내 방어막 증가
```

#### 회복 카드

```
⚔️ = 0, 🛡️ = 0
효과: HEAL
→ 내 HP 회복
```

#### 복합 카드

```
⚔️ > 0, 🛡️ = 0
효과: HEAL, SHIELD
→ 피해 + 회복/방어막
```

## ✅ 성공 확인 체크리스트

### 카드 표시

- [ ] ⚔️ 숫자 = 설명의 피해량
- [ ] 설명 읽기 편함
- [ ] 실제 피해 = 표시 피해

### 싱글 플레이

- [ ] 카드 사용 → 보스 HP 정확히 감소
- [ ] 회복 카드 → 내 HP 증가
- [ ] 방어 카드 → 실드 생성

### PvP

- [ ] 선공 공격 → 후공 HP 감소 ✅
- [ ] 후공 공격 → 선공 HP 감소 ✅
- [ ] 에너지: 라운드 단위 증가 ✅
- [ ] 나가기 → 상대에게 알림 ✅
