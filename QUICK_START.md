## 🚀 지금 바로 실행하기

### 1️⃣ 싱글 플레이 (AI 보스 대전)

```powershell
npm run dev
```

브라우저: `http://localhost:5173`

---

### 2️⃣ 멀티플레이 (PvP 대전) ⭐

```powershell
npm run dev:full
```

그 다음:

1. **일반 브라우저 창**: `http://localhost:5173`
   - 멀티플레이 클릭
   - 🟢 **서버 연결됨** 확인 (좌측 상단)
   - 닉네임: `플레이어1`, 방 이름: `테스트방` 입력
   - "방 생성" 클릭

2. **시크릿 모드 새 창** (Ctrl + Shift + N):
   - `http://localhost:5173` 접속
   - 멀티플레이 클릭
   - 닉네임: `플레이어2` 입력
   - "테스트방" → "참가하기" 클릭

3. 두 창에서 각각 **"준비 완료"** 클릭

4. **게임 시작!** ✅

## 🐛 자주 묻는 질문

### Q: "방 생성" 버튼이 비활성화되어 있어요

**A**: 백엔드 서버 미실행 → `npm run dev:full` 실행

---

### Q: 카드 이미지가 이모지 대신 실제 그림을 쓰고 싶어요

**A**: `CUSTOM_CARD_IMAGES.md` 파일 참고

---

### Q: 혼자서 PvP를 테스트하려면?

**A**: 
1. 일반 브라우저 창 + 시크릿 모드 창 사용
2. 또는 Chrome + Edge 각각 사용
3. 자세한 방법: `PVP_TEST_GUIDE.md`

---

### Q: 다른 사람과 실제로 PvP 하려면?

**A**: 
- 같은 Wi-Fi 네트워크 필요
- `src/hooks/useSocket.ts`에서 서버 IP 수정
- 자세한 방법: `PVP_TEST_GUIDE.md` 참고

## 🎮 게임 플레이 팁

### 싱글 플레이

1. **카드 효과 확인**: 
   - 💚 HEAL = 체력 회복
   - 🛡️ SHIELD = 방어막

   - 🔥 DOT = 지속 피해

   - 후반: 고코스트 강력 카드 사용

3. **에너지 관리**:
