## 2️⃣ PVP 패배 역전 시스템

### 특징
- 😢 **PVP 패배 시** 오픈뱅킹 팝업 자동 표시
- 🏦 **신한은행 특화** 4가지 옵션
- 💎 **히든 카드** 획득으로 역전의 기회

### 오픈뱅킹 옵션 (신한은행 특화)

**1. 타행 IRP → 신한은행 이전**
```
타 은행의 개인형 퇴직연금을 신한은행으로 이전
혜택: 잔액 1천만원 + 우대 금리 6.8%
```

**2. 자동 정기결제 → 신한은행 변경**
```
각종 자동이체/정기결제를 신한은행 계좌로 변경
혜택: 월 100만원 자동이체
```

**3. 추가 신한은행 상품 가입**
```
신한 주택청약종합저축 또는 연금저축 가입
혜택: 청약 100만원 + 연금 500만원
```

**4. 청약상품 → 신한은행 이전**
```
타 은행의 청약통장을 신한은행으로 이전
혜택: 청약 횟수 유지 + 우대
```

## 3️⃣ 밸런스 조정된 히든 카드

### 🏦 신한 금융의 힘

```
┌─────────────────────────────────┐
│ 🏦 신한 금융의 힘 (LEGENDARY)    │
├─────────────────────────────────┤
│ 코스트: 4                        │
│ 공격력: 8                        │
│ 방어력: 3                        │
├─────────────────────────────────┤
│ 신한은행 통합 금융의 힘!         │
│                                  │
│ • 상대에게 8 피해                │
│ • 내 체력 5 회복                 │
│ • 방어막 3 생성                  │
│ • 다음 턴 에너지 +2              │
├─────────────────────────────────┤
│ 💎 밸런스 잡힌 역전의 카드       │
└─────────────────────────────────┘
```

**밸런스 분석**:
- 총 가치: 코스트 10 상당
- 사용 비용: 코스트 4
- **평가**: 강력하지만 게임을 즉시 뒤집을 정도는 아님

## 🎮 전체 게임 플로우

### A. 신규 고객 플로우

```
로그인 화면
  ↓
"신규 고객입니다" 클릭
  ↓
이름 입력: 홍길동
  ↓
계좌 개설 팝업
  - 이메일/비밀번호
  - 신한은행 상품 3종 선택
  ↓
계좌 개설 완료
  - 스타터 덱 30장 자동 지급
  ↓
메인 화면
  - 싱글 플레이 or PvP 선택
```

---

### B. 기존 고객 플로우

```
로그인 화면
  ↓
기존 계정 로그인
  (염승훈, 이태영, 좌상목, 서재만, 강승완, 배형준, 임지훈)
  ↓
메인 화면
  - 거래 패턴 기반 덱 (25~40장)
  - 싱글 플레이 or PvP 선택
```

---

### C. Guest 플로우

```
로그인 화면
  ↓
"🎮 Guest로 시작하기" 클릭
  ↓
메인 화면
  - 기본 덱 사용
  - 싱글 플레이만 가능
  - PvP 시도 → "로그인 필요" 안내
```

---

### D. PvP 패배 역전 플로우

```
PvP 전투
  ↓
패배 (HP 0)
  ↓
게임 종료 모달
  😢 패배
  "더 강해져서 돌아오세요"
  ↓
🏦 신한은행 추가 상품으로 역전하기 (클릭)
  ↓
오픈뱅킹 팝업
  - 4가지 옵션 선택
  - 연동하기
  ↓
히든 카드 획득
  🏦 신한 금융의 힘 (LEGENDARY)
  ↓
재도전
  - 히든 카드 사용
  - 역전 승리! 🎉
```
