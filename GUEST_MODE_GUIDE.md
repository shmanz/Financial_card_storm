# 🎮 Guest 모드 사용 가이드

## ✅ Guest 모드란?

**로그인 없이 바로 게임을 체험**할 수 있는 모드입니다!
- 회원가입 없이 즉시 플레이
- 200개의 랜덤 거래 데이터로 카드 생성
- 싱글 플레이 완전 지원
- 언제든지 회원가입 가능

---

## 🚀 Guest로 시작하기

### 방법 1: 첫 화면에서 바로 시작

1. 게임 실행: `npm run dev:full`
2. 브라우저: `http://localhost:5173`
3. 로그인 화면에서 **"🎮 Guest로 시작하기"** 버튼 클릭
4. 바로 게임 시작! ✅

### 방법 2: 기존 회원 로그인

- 이름: 염승훈, 이태영, 좌상목, 서재만, 강승완, 배형준, 임지훈
- 비밀번호: `password123`

---

## 🎯 Guest로 할 수 있는 것

### ✅ 가능한 기능

1. **싱글 플레이**
   - AI 보스와 전투
   - 200개 거래 데이터 기반 카드 덱
   - 모든 카드 효과 사용 가능

2. **거래 내역 확인**
   - 내 거래 내역 버튼 클릭
   - Guest 전용 랜덤 거래 200건 확인

3. **오픈뱅킹 연동**
   - 패배 시 오픈뱅킹 연동 가능
   - 히든 카드 획득 가능
   - 타 은행 상품 가져오기

4. **자유로운 체험**
   - 모든 싱글 플레이 기능
   - 무제한 재시작
   - 다양한 카드 조합 테스트

### ❌ 제한되는 기능

1. **멀티플레이 (PvP)**
   - Guest는 PvP 불가
   - 시도 시 로그인 유도 모달 표시
   - 회원가입 후 이용 가능

---

## 🔄 Guest에서 회원으로 전환

### 방법 1: 헤더 버튼 이용

1. 화면 우측 상단 **"회원가입하기"** 버튼 클릭
2. 로그인 화면으로 이동
3. **"신규 고객입니다"** 선택
4. 이름 입력 후 계좌 개설

### 방법 2: PvP 시도 시

1. 메인 화면에서 **"멀티플레이 (PvP)"** 클릭
2. 로그인 필요 모달 표시
3. **"회원가입하기"** 클릭
4. 계좌 개설 후 PvP 이용

---

## 💡 Guest 모드 특징

### 자동 생성되는 것들

1. **거래 내역 200건**
   - 10개 카테고리 랜덤 분포
   - FOOD, CAFE, SHOPPING, TRAVEL 등
   - 최근 90일 이내 거래

2. **기본 은행 상품**
   - 체험용 통장 (잔액 300만원)
   - 체험용 카드 (한도 200만원)

3. **카드 덱**
   - 거래 패턴 기반 자동 생성
   - 50종 카드 능력 중 랜덤 조합
   - 매번 다른 덱 구성 가능

### Guest 데이터 저장

- ✅ LocalStorage에 세션 저장
- ✅ 새로고침해도 유지됨
- ⚠️ 브라우저 캐시 삭제 시 초기화
- ⚠️ 회원가입하면 Guest 데이터는 사라짐

## 🆘 FAQ

### Q: Guest 데이터가 사라졌어요!

**A**: 브라우저 캐시를 삭제했거나, 시크릿 모드를 사용했을 수 있습니다.
- 일반 모드에서는 LocalStorage에 저장됨
- 시크릿 모드는 종료 시 데이터 삭제

### Q: Guest에서 회원가입하면 데이터가 이어지나요?

**A**: 아니요, 새로운 회원 계정으로 시작합니다.
- Guest 데이터는 체험용
- 회원가입 시 새로운 거래 패턴 생성

### Q: Guest로 히든 카드 획득 가능한가요?

**A**: 네! 가능합니다.
- 보스 패배 → 오픈뱅킹 연동
- 히든 카드 "💎 금융의 신" 획득
- Guest도 동일한 기능 사용 가능

### Q: Guest로 PvP 못하나요?

**A**: 회원만 가능합니다.
- 멀티플레이는 계정 필요
- Guest는 싱글 플레이 전용
- 회원가입 후 PvP 이용하세요
